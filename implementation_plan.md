# ことのは ランディングページ 実装計画

## 概要

「ことのは」の公式ランディングページを、Next.js App Router + Tailwind CSS + GSAP で構築します。
添付のデザインモックアップに忠実に、「言葉」「文学」「温かみ」「静寂」「内省」をコンセプトとした美しいLPを実装します。

## ユーザー確認事項

> [!IMPORTANT]
> **Tailwind CSS のバージョン**: Tailwind CSS **v4（最新）** を使用する想定です。v3 を希望される場合はお知らせください。

> [!IMPORTANT]
> **画像アセット**: デザインモックアップに含まれるイラスト（万年筆、ノート、ラジオ、本とペン、対話する人々、グループワーク、ガーランド、窓辺の風景など）は、手書き風線画のプレースホルダーSVGとして実装します。後日、実際のアセットに差し替え可能な構成とします。

> [!NOTE]
> **GSAP ライセンス**: GSAP は非営利・個人利用であればフリーです。ScrollTrigger プラグインも同様に使用します。

### アニメーション (GSAP)

- **基本パターン**: 各セクション要素が `ScrollTrigger` でビューポートに入ると `y: 30` → `y: 0`, `opacity: 0` → `opacity: 1` でフェードイン
- **イージング**: `power2.out`（柔らかく自然）
- **デュレーション**: 0.8〜1.2秒（静寂さを保つ）
- **スタガー**: Activities セクションのカードは 0.15秒 の間隔でスタガー
- **`"use client"` ディレクティブ**: GSAP を使用するコンポーネントにはクライアントコンポーネント宣言が必要

### レスポンシブ対応

| ブレークポイント | 対応内容 |
|---|---|
| `>= 1024px` (PC) | フルレイアウト（左右分割、5列グリッドなど） |
| `768px〜1023px` (タブレット) | 2〜3列グリッド、一部縦積み |
| `< 768px` (SP) | 1列縦積み、ハンバーガーメニュー |

## 実行するコマンド一覧

```bash
# 1. Next.js プロジェクト初期化
npx -y create-next-app@latest ./ --typescript --tailwind --eslint --app --src-dir=false --import-alias="@/*" --turbopack --yes

# 2. GSAP インストール
npm install gsap

# 3. 開発サーバー起動（最終確認時）
npm run dev
```
